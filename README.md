✋ Realtime Hand Gesture Recognition System


📌 Overview

This project is a real-time hand gesture recognition system built using Computer Vision and Machine Learning techniques. It detects and classifies hand gestures using a webcam and enables touchless interaction with computers.

The system uses MediaPipe for hand tracking and OpenCV for image processing, making it fast, accurate, and efficient.

🚀 Features


🎯 Real-time hand detection using webcam


✋ Accurate gesture recognition


⚡ Fast and lightweight implementation


🧠 Uses MediaPipe for hand landmarks


💻 Easy to integrate with other applications


🎮 Can be extended for gaming, automation, or navigation


🛠️ Tech Stack

Python 🐍

OpenCV

MediaPipe

NumPy

⚙️ Installation

1️⃣ Clone the repository
git clone (link)....

cd Realtime-Hand-Gesture-Recognition

2️⃣ Install dependencies
pip install -r requirements.txt

Or manually:

pip install opencv-python mediapipe numpy
▶️ Usage

Run the main script:

python main.py

📸 Your webcam will open and start detecting hand gestures in real-time.
